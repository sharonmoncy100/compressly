Resize & compress to passport size photo (35 x 45 mm / 2 x 2 in)

Practical guide for creating passport and visa photos that meet size and file-weight requirements for Indian portals. Fast, private, in-browser.

Many Indian passport and visa portals ask for a passport photo with specific physical size (35 x 45 mm or 2 x 2 inch) and a file-size limit. The exact pixel dimensions and allowed KB vary by portal — check the portal instructions first. The steps below give safe, commonly accepted values and how to produce them with Compressly.

Common physical & digital examples

  • Physical print: 35 x 45 mm (common for Indian passport)
  • Common digital pixel examples (approx.):
    • 35 x 45 mm at 300 DPI ≈ 413 x 531 px (good-quality upload)
    • 2 x 2 inch at 300 DPI ≈ 600 x 600 px
    • Many portals accept 350 x 450 px — check portal recommendations
  • Typical file-size limits: many portals require ≤100 KB; some allow up to 200 KB. Very old portals may ask for ≤50 KB. Always check the portal guidelines before final upload.
  • Preferred formats: JPG / JPEG (recommended), PNG only if explicitly required, WebP when the portal supports modern formats.

How to create a passport-size image that meets portal rules

  1. Crop to the required physical/digital aspect ratio (35:45). If the portal gives pixel values, use those exactly.
  2. Resize to the target pixel dimensions (for example 413 x 531 px or 600 x 600 px at 300 DPI) using an image editor or Compressly's downscale options.
  3. Choose JPEG (or WebP if supported). Use the Target (KB) box with a safe value — for many Indian portals try 100 KB.
  4. Compress and inspect the image visually. If quality is OK, download and upload to the portal. If rejected, slightly reduce target KB or downscale further.
Quick practical rule: If you need to be conservative, produce both a high-quality copy (keep original backup) and a compressed copy (100 KB or smaller) for upload.

Portal examples & notes (always check the portal)

Different portals use different rules. For example, a university or government portal may require a small file (≤100 KB) while passport/visa agencies may ask for higher resolution. When in doubt, follow the portal instructions and use our Target (KB) box to match the requested size.

Q: Which image format is safest for passport uploads?
A: JPEG is the safest and most widely accepted. Use WebP only if the portal explicitly supports it.

Q: Should I use high DPI (300) for digital upload?
A: DPI matters for printing. For online uploads, pixel dimensions are important. Use higher pixel dimensions if the portal asks for high resolution.
