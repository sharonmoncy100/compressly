Compress image for exam forms (JEE, NEET, university portals)

Reduce image file size to meet the KB limits required by Indian exam portals. Fast, private and browser based.

Exam portals usually require a passport style photo and sometimes a signature image. These portals often reject files that exceed the allowed file size, even if the image looks visually correct.

Common file size rules used by exam portals

  • File format: JPEG or JPG is accepted by most exam systems.
  • File size limits: Common limits range from 50 KB to 200 KB. Some portals allow up to 500 KB.
  • Background and dimensions: These must already match the portal instructions before compression.

How to prepare your image before compression

  1. Make sure the photo already meets the required background and dimensions.
  2. Upload the image to Compressly and choose a target KB limit.
  3. Download the compressed image and upload it to the exam portal.
  4. If rejected, try a slightly lower KB target.
Important: Compressly only reduces file size. It does not change image dimensions or background.

FAQ

Q: My image meets the KB limit but was rejected. Why?
A: Many portals also check background, clarity and dimensions. Make sure these match the portal rules.

Q: Can I use Compressly for signature images?
A: Yes. Use JPG if transparency is not required. If transparency is required, check whether the portal accepts PNG.
